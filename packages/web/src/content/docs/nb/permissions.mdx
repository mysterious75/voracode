## Standarder

Hvis du ikke spesifiserer noe, starter Voracode fra tillatte standardinnstillinger:

- De fleste tillatelser er satt til `"allow"` som standard.
- `doom_loop` og `external_directory` er standard til `"ask"`.
- `read` er `"allow"`, men `.env` filer nektes som standard:

```json title="voracode.json"
{
  "permission": {
    "read": {
      "*": "allow",
      "*.env": "deny",
      "*.env.*": "deny",
      "*.env.example": "allow"
    }
  }
}
```
