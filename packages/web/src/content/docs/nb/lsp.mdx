## Konfigurasjon

Du kan aktivere og tilpasse LSP-servere gjennom `lsp`-delen i Voracode-konfigurasjonen.

For 횄짜 aktivere alle innebygde LSP-servere, sett `lsp` til `true`.

```json title="voracode.json"

  "$schema": "https://voracode.ai/config.json",
  "lsp": true
}
```

Bruk et objekt for 횄짜 beholde innebygde servere aktivert mens du konfigurerer overstyringer eller egendefinerte servere.

```json title="voracode.json"
{
  "$schema": "https://voracode.ai/config.json",
  "lsp": {}
}
```

| ---------------- | -------- | -------------------------------------------------------- |
| `disabled`       | boolean  | Sett dette til `true` for 횄짜 deaktivere LSP-serveren      |
| `command`        | string[] | Kommandoen for 횄짜 starte LSP-serveren                     |
| `extensions`     | string[] | Filendelser denne LSP-serveren skal h횄짜ndtere             |
| `env`            | object   | Milj횄쨍variabler som skal angis n횄짜r serveren startes       |
| `initialization` | object   | Initialiseringsalternativer for 횄짜 sende til LSP-serveren |

La oss se p횄짜 noen eksempler.

### Milj횄쨍variabler

Bruk egenskapen `env` til 횄짜 angi milj횄쨍variabler n횄짜r du starter LSP-serveren:

### Initialiseringsalternativer

Bruk egenskapen `initialization` til 횄짜 sende initialiseringsalternativer til LSP-serveren. Dette er serverspesifikke innstillinger sendt under LSP `initialize`-foresp횄쨍rselen:

```json title="voracode.json" {5-9}

        "preferences": {

        }
      }
    }
  }
}
```

:::note
Initialiseringsalternativer varierer etter LSP-server. Sjekk dokumentasjonen til LSP-serveren for tilgjengelige alternativer.
:::

### Deaktivering av LSP-servere

Hvis `lsp` utelates, er alle LSP-servere deaktivert. For 횄짜 deaktivere alle LSP-servere etter at en annen konfigurasjon har aktivert dem, sett `lsp` til `false`:

```json title="voracode.json" {3}
{
  "$schema": "https://voracode.ai/config.json",
  "lsp": false
}
```

For 횄짜 deaktivere en **spesifikk** LSP-server, sett `disabled` til `true`:

```json title="voracode.json" {5}
{
  "$schema": "https://voracode.ai/config.json",
  "lsp": {
    "typescript": {
      "disabled": true
    }
  }
}
```
