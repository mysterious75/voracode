## Opprette et verkt횄쨍y

Verkt횄쨍y er definert som **TypeScript**- eller **JavaScript**-filer. Verkt횄쨍ydefinisjonen kan imidlertid p횄짜kalle skript skrevet p횄짜 **alle spr횄짜k** - TypeScript eller JavaScript brukes bare for selve verkt횄쨍ydefinisjonen.

### Plassering

De kan defineres:

- Lokalt ved 횄짜 plassere dem i `.voracode/tools/`-katalogen til prosjektet ditt.
- Eller globalt, ved 횄짜 plassere dem i `~/.config/voracode/tools/`.

### Struktur

Den enkleste m횄짜ten 횄짜 lage verkt횄쨍y p횄짜 er 횄짜 bruke `tool()`-hjelperen som gir typesikkerhet og validering.

```ts title=".voracode/tools/database.ts" {1}

  description: "Query the project database",
  args: {
    query: tool.schema.string().describe("SQL query to execute"),
  },
  async execute(args) {
    // Your database logic here
    return `Executed query: ${args.query}`

```

**filnavnet** blir **verkt횄쨍ynavnet**. Ovennevnte oppretter et `database`-verkt횄쨍y.

#### Flere verkt횄쨍y per fil

Du kan ogs횄짜 eksportere flere verkt횄쨍y fra en enkelt fil. Hver eksport blir **et separat verkt횄쨍y** med navnet **`<filename>_<exportname>`**:

```ts title=".voracode/tools/math.ts"

export const add = tool({
  description: "Add two numbers",
  args: {
    a: tool.schema.number().describe("First number"),
    b: tool.schema.number().describe("Second number"),
  },
  async execute(args) {
    return args.a + args.b
  },
})

export const multiply = tool({
  description: "Multiply two numbers",
  args: {
    a: tool.schema.number().describe("First number"),
    b: tool.schema.number().describe("Second number"),
  },
  async execute(args) {
    return args.a * args.b
  },
})
```

Dette lager to verkt횄쨍y: `math_add` og `math_multiply`.

#### Navnekollisjoner med innebygde verkt횄쨍y

Egendefinerte verkt횄쨍y er n횄쨍klet etter verkt횄쨍ynavn. Hvis et egendefinert verkt횄쨍y bruker samme navn som et innebygd verkt횄쨍y, vil det egendefinerte verkt횄쨍yet ha forrang.

For eksempel erstatter denne filen det innebygde `bash`-verkt횄쨍yet:

```ts title=".voracode/tools/bash.ts"

  description: "Restricted bash wrapper",
  args: {
    command: tool.schema.string(),
  },
  async execute(args) {
    return `blocked: ${args.command}`
  },
})
```

:::note
Foretrekk unike navn med mindre du med vilje 횄쨍nsker 횄짜 erstatte et innebygd verkt횄쨍y. Hvis du vil deaktivere et innebygd verkt횄쨍y, men ikke overstyre det, bruk [tillatelser](/docs/permissions).
:::

### Argumenter

Du kan bruke `tool.schema`, som bare er [Zod](https://zod.dev), for 횄짜 definere argumenttyper.

```ts "tool.schema"

  async execute(args, context) {
    // Tool implementation
    return "result"
  },
}
```

### Kontekst

Verkt횄쨍y mottar kontekst om gjeldende 횄쨍kt:

```ts title=".voracode/tools/project.ts" {8}
import { tool } from "@voracode-ai/plugin"

export default tool({
  description: "Get project information",
  args: {},
  async execute(args, context) {
    // Access context information
    const { agent, sessionID, messageID, directory, worktree } = context
    return `Agent: ${agent}, Session: ${sessionID}, Message: ${messageID}, Directory: ${directory}, Worktree: ${worktree}`
  },
})
```

Bruk `context.directory` for 횄쨍ktens arbeidskatalog.
Bruk `context.worktree` for git-arbeidstreet.

## Eksempler

### Skrive et verkt횄쨍y i Python

Du kan skrive verkt횄쨍yene dine p횄짜 hvilket som helst spr횄짜k du vil. Her er et eksempel som legger til to tall ved hjelp av Python.

F횄쨍rst lager du verkt횄쨍yet som et Python-skript:

```python title=".voracode/tools/add.py"

a = int(sys.argv[1])
b = int(sys.argv[2])
print(a + b)
```

Lag deretter verkt횄쨍ydefinisjonen som p횄짜kaller den:

```ts title=".voracode/tools/python-add.ts" {10}
import { tool } from "@voracode-ai/plugin"
import path from "path"

export default tool({
  description: "Add two numbers using Python",
  args: {
    a: tool.schema.number().describe("First number"),
    b: tool.schema.number().describe("Second number"),
  },
  async execute(args, context) {
    const script = path.join(context.worktree, ".voracode/tools/add.py")
    const result = await Bun.$`python3 ${script} ${args.a} ${args.b}`.text()
    return result.trim()
  },
})
```

Her bruker vi [`Bun.$`](https://bun.com/docs/runtime/shell)-verkt횄쨍yet for 횄짜 kj횄쨍re Python-skriptet.
