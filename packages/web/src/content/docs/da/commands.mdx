## Oprettelse af kommandofiler

Opret markdown-filer i mappen `commands/` for at definere brugerdefinerede kommandoer.

Opret `.voracode/commands/test.md`:

```md title=".voracode/commands/test.md"
---
description: Run tests with coverage
agent: build
model: anthropic/claude-3-5-sonnet-20241022
---

Run the full test suite with coverage report and show any failures.
Focus on the failing tests and suggest fixes.
```

Frontmatter definerer kommandoegenskaber. Indholdet bliver skabelonen.

Brug kommandoen ved at skrive `/` efterfulgt af kommandonavnet.

```bash frame="none"
"/test"
```

## Konfiguration

Du kan tilf횄쨍je brugerdefinerede kommandoer gennem Voracode-konfigurationen eller ved at oprette markdown-filer i `commands/`-mappen.

---

### JSON

Brug indstillingerne `command` i din Voracode [config](/docs/config):

```json title="voracode.jsonc" {4-12}
{
  "$schema": "https://voracode.ai/config.json",
  "command": {
    // This becomes the name of the command
    "test": {
      // This is the prompt that will be sent to the LLM
      "template": "Run the full test suite with coverage report and show any failures.\nFocus on the failing tests and suggest fixes.",
      // This is shown as the description in the TUI
      "description": "Run tests with coverage",
      "agent": "build",
      "model": "anthropic/claude-3-5-sonnet-20241022"
    }
  }
}
```

Nu kan du k횄쨍re denne kommando i TUI:

```bash frame="none"
/test
```
