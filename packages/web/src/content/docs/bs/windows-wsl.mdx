Iako Voracode moze raditi direktno na Windowsu, preporucujemo [Windows Subsystem for Linux (WSL)](https://learn.microsoft.com/en-us/windows/wsl/install) za najbolje iskustvo. WSL daje Linux okruzenje koje glatko radi sa Voracode funkcijama.

WSL nudi bolje performanse fajl sistema, punu terminalsku podrsku i kompatibilnost s razvojnim alatima na koje se Voracode oslanja.
:::

## Postavljanje

<Steps>

1. **Instalirajte WSL**

   Ako vec niste, [instalirajte WSL](https://learn.microsoft.com/en-us/windows/wsl/install) prema zvanicnom Microsoft vodicu.

2. **Instalirajte Voracode u WSL-u**

   Kad je WSL spreman, otvorite WSL terminal i instalirajte Voracode jednom od [metoda instalacije](/docs/).

   ```bash
   curl -fsSL https://voracode.ai/install | bash
   ```

3. **Koristite Voracode iz WSL-a**

   Idite u direktorij projekta (Windows fajlovima pristupate preko `/mnt/c/`, `/mnt/d/` itd.) i pokrenite Voracode.

   ```bash
   cd /mnt/c/Users/YourName/project
   voracode
   ```

</Steps>

## Desktop aplikacija + WSL Server

Ako preferirate Voracode Desktop aplikaciju, ali zelite da server radi u WSL-u:

1. **Pokrenite server u WSL-u** sa `--hostname 0.0.0.0` da dozvolite vanjske konekcije:

   ```bash
   voracode serve --hostname 0.0.0.0 --port 4096
   ```

2. **Povezite Desktop aplikaciju** na `http://localhost:4096`

:::note
Ako `localhost` ne radi u vasem setupu, povezte se preko WSL IP adrese (u WSL-u: `hostname -I`) i koristite `http://<wsl-ip>:4096`.
:::

:::caution
Kada koristite `--hostname 0.0.0.0`, postavite `VORACODE_SERVER_PASSWORD` da zastitite server.

```bash
VORACODE_SERVER_PASSWORD=your-password voracode serve --hostname 0.0.0.0
```

:::

## Web klijent + WSL

Za najbolje web iskustvo na Windowsu:

1. **Pokrenite `voracode web` u WSL terminalu** umjesto u PowerShell-u:

   ```bash
   voracode web --hostname 0.0.0.0
   ```

2. **Otvorite iz Windows browsera** na `http://localhost:<port>` (Voracode ispisuje URL)

Pokretanje `voracode web` iz WSL-a osigurava ispravan pristup fajl sistemu i terminalsku integraciju, a i dalje je dostupno iz Windows browsera.

## Savjeti

- Drzite Voracode u WSL-u za projekte na Windows diskovima - pristup fajlovima je jednostavan
- Koristite VS Code [WSL ekstenziju](https://code.visualstudio.com/docs/remote/wsl) uz Voracode za integrisan tok rada
- Vase Voracode konfiguracije i sesije cuvaju se u WSL okruzenju na `~/.local/share/voracode/`
